# Florida Asthma Resource Kit

A comprehensive, version-controlled repository of educational materials and resources specifically designed for asthma patients living in Florida.

## Purpose

This resource kit addresses the unique challenges faced by asthma patients in Florida, including hurricane preparedness, high humidity, mold exposure, and distinct pollen seasons. By using a GitHub repository, we ensure that all materials are professionally reviewed, updated, and easily accessible to healthcare providers and patients alike.

## Table of Contents

1. **Asthma Action Plans** – How to use and update your personalized asthma action plan.
2. **Hurricane Preparedness Checklist** – Essential steps for asthma patients before, during, and after a hurricane.
3. **Managing Mold-Induced Asthma in Humid Climates** – Information on identifying and reducing mold exposure in Florida homes.
4. **Asthma Control Test (ACT) Score Tracker** – A template for tracking your ACT scores over time to monitor asthma control.

## How to Use This Kit

- **Patients**: Download the PDFs or view the guides directly from this repository. Each guide is written in plain language with actionable steps.
- **Healthcare Providers**: Use these materials as part of patient education, or contribute your own resources via pull requests.
- **Contributors**: See the [Contributing Guidelines](CONTRIBUTING.md) for how to propose changes or add new resources.
